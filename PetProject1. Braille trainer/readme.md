Тренажер Брайля

...::: Описание  :::...
Тренажер Брайля впредназначен для того, чтобы вы могли в игровой форме освоить шрифт Брайля (русский алфавит).


...:::  Как играть  :::...

Запускать файл BEGIN.py в среде PyCharm (или другой, любимой вами IDE).

Открывается окно с приветствием. Здесь можно увидеть весь русский алфавит в шрифте Брайля.
После ознакомления с текстом приветствия закройте окно, после этого появится 
основное окно приложения, на котором расположены кнопки для выбора нужного действия.


Кнопка ["Начать..."] позволяет перейти к выбору режима игры.

Перед началом игры в любом режиме потребуется представиться (ввести имя игрока в окне вверху справа), 
и нажать кнопку ["Начать!"]. До нажатия на кнопку виджеты в центральной части формы неактивны. 

  Режимы игры:
  ["Буквы"] - показывается случайная буква русского алфавита, вам надо вспомнить ее код в шрифте Брайля. 
    Код буквы набирается с помошью нажатия на кнопки с цифрами 1, 2, 3, 4, 5 и 6. 
    Одно нажатие - точка становится черной ("выпуклой"), еще одно нажатие - точка становится белой ("пустой"). 
    После того, как вы наберете нужную комбинацию точек, нажмите кнопку ["Проверить"]. Над кнопкой появится соответствующее сообщение.
    Буква поменяется, и вы сможете играть дальше.
    Счет в правой части окна показывает текущий счет: количество правильно угаданных букв и количество ошибок.

  ["Точки"] - показывается код в шрифте Брайля для некоторой буквы русского алфавита. Вам надо нажать кнопку с соответствующей буквой. 
    Если ваш выбор верен, то кнопка станет зеленого цвета, если вы ошиблись - кнопка станет красной.
    Цвет букв сохраняется на протяжении всей игы, чтобы вы могли видеть, какие буквы вы лучше запомнили, а в каких часто ошибаетесь.
    Счет в правой части окна показывает общее количество правильно угаданных букв и количество ошибок.
    На ранних этапах изучения можно воспользоваться подсказкой (внизу справа). 

Чтобы закончить игру - достаточно закрыть окно. При этом режим игры, имя игрока, текущей счет, дата и время будут сохранены в базе данных.
И снова откроется основное окно приложения с кнопками для выбора нужного действия.


Кнопка ["Все попытки"] позволяет просмотреть итоги предыдущих игр. Записи из базы данных будут показаны здесь же, в виде таблицы. 
Убрать таблицу можно, нажав кнопку ["Закрыть"].


Кнопки ["Факт дня"] и ["О программе"] открывают новое окно, в котором загружен соответствующий текст. 
Здесь тоже можно увидеть весь русский алфавит в шрифте Брайля.
Все тексты, в том числе текст приветствия, находятся в базе данных, откуда загружаются по необходимости. 
"Приветствие" - запись № 0, "О программе" - запись № 999, "Факт дня" - случайная запись из оставшихся. 
Закрывая окно, возвращаемся в основное окно приложения.


Кнопка ["Выход"] заканчивает работу программы. 


...:::  Технические особенности  :::...

Тексты для факта дня, коды символов для шрифта Брайля находятся в базе данных, куда их можно добавлять.
Поэтому данный тренажер может быть легко расширен для изучения латинского алфавита и цифр.


...:::  Структура проекта  :::...

BEGIN.PY		файл исходного кода, начало проекта
game.py			файл исходного кода, доп.файл, содержит классы с играми
braille.db		база данных, содержит три таблицы (facts - тексты, results - результаты игр, 
				braille - таблица символов и их кодов)
main.iu			интерфейс основного окна приложения с кнопками
choice.ui		интерфейс вспом.окна для выбора режима игры
braille.png		картинка - подложка в окне с фактом дня
icon.png		иконка приложения
requirements.txt  	описывает зависимости проекта 
readme.md		данный файл
