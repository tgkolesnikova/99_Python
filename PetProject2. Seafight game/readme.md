******************** М О Р С К О Й   Б О Й ***********************

Симулятор игрального автомата времен СССР "Морской бой"

Стоимость игры - 15 копеек.
Количество выстрелов - 10 штук.


****** С Ю Ж Е Т ******

Перед началом игры попросите у мамы монетку, затем засуньте её (монетку, не маму!) в монетоприёмник игрального автомата. 
Перед вами появится морская гладь, по которой ходят суда вражеского флота.
Вы - капитан подлодки. Ваша задача - поразить корабли противника.

Используя клавиши <Стрелка влево> <Стрелка вправо>, перемещайте видоискатель перископа для прицеливания.
Чтобы произвести выстрел - нажмите клавишу <Пробел> (ещё раз стрелять из того же положения не получается, 
т.к. действует отдача торпедного аппарата))) Понажимайте стрелки, и выстрел станет возможен).

После 10-го выстрела игра завершается.
Количество поражённых кораблей противника будет показано на табло в конце игры.
